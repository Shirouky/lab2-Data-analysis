# Лабораторная работа №2: Парсинг интернет-магазина

## Описание
Парсинг данных с сайта biser-optom.ru - сбор информации о товарах (бисер Preciosa).

## Технологии
- **Python 3.7+**
- **requests** - HTTP-запросы
- **BeautifulSoup4** - парсинг HTML

## Структура проекта
```
LR2.ipynb           # основной скрипт
beads.json          # результаты (генерируется)
Saved_pages/        # HTML-страницы
```

## Функциональность

### Задание 1: Скачивание данных
- **Источник**: biser-optom.ru/category/biser-Preciosa
- **Объем**: 79 страниц (>500 товаров)
- **Настройки**:
  - User-Agent для обхода защиты
  - Таймаут 10 секунд
  - Сохранение HTML в папку Saved_pages

### Задание 2: Парсинг данных
**Извлекаемые параметры (14 полей):**
- Основные: id, id_web, code, type, size, color
- Цены: price, old_price, currency, sale
- Дополнительно: weight, quantity, image, url

## Особенности реализации

### Класс Card
```python
class Card():
    # Парсинг карточки товара
    def set_price(), set_quantity(), export(), etc.
```

### Декораторы
- `@decorator` - обработка ошибок → None
- `@digit` - преобразование строк в числа
